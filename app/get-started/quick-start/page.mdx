## Quick Start Guide

### Step 1 : Download and Install

1. Go to [incerto.in](https://incerto.in) and click the "Download" button. The supported OS are:
   - **Windows:** `.exe` installer
   - **macOS (Apple Silicon):** `.dmg` file
2. Run the installer and follow the on-screen instructions.  
3. Launch Incerto after installation.

### Step 2 : Adding Your Database

1. Open Incerto and navigate to **Connections**.
2. Click **Add Database** and fill in:
   - **Database Type:** e.g., PostgreSQL, MySQL, ClickHouse
   - **Host / Port**
   - **Username & Password**
3. Click **Test Connection** to ensure everything works, then **Save**.

### Step 3 : Task to SQL

1. In the Incerto interface, go to the **AI Task** panel.
2. Type a natural language request like:
   > “Find the slowest queries in the last 24 hours”
3. Incerto automatically generates and runs the SQL, showing you the results.

### Step 4 : See Current Firing Problems

1. Navigate to the **Issues** dashboard.
2. View real-time **firing alerts** such as:
   - High CPU usage
   - Query timeouts
   - Lock contention
3. Click any issue for **AI-recommended remediation steps**.
